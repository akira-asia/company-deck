# Proposal Review Prompt

完成資料をレビューし、改善点を抽出するためのプロンプトです。

## 観点

- 意思決定に必要な情報が揃っているか
- 主張と根拠に飛躍がないか
- ストーリーに一貫性があるか
- デザインが理解を妨げていないか
- 次アクションが明確か
- 専門用語が経営指標や意思決定論点に翻訳されているか
- 施策ごとの担当部門、評価KPI、運用負荷が明確か
- ROIに売上増分だけでなく、粗利、手数料削減、運用工数が含まれているか

## 出力形式

- 重大な改善点
- 軽微な改善点
- 追加すると効果的な要素
- このまま使える強み
- 次回以降の共通ルールに昇格すべき学び
