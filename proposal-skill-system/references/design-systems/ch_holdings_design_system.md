## 3. タイポグラフィ

### 3.1 フォントファミリー

```css
font-family: 'Hiragino Sans', 'Yu Gothic UI', 'Noto Sans JP', sans-serif;
```

明朝は使わない。太字傾向のゴシックで誠実さと切れ味を出す。

### 3.2 タイプスケール

| ロール | サイズ | Weight | 用途 |
|---|---|---|---|
| Title Slide | `44-56px` | 700 | 表紙・章扉 |
| Slide Title | `26px` | 700 | 各スライドのタイトル |
| Lead | `18px` | 600 | スライド冒頭のリード文 |
| Body | `15px` | 400 | 本文 |
| Caption | `11-13px` | 400 | フッター・脚注・出典 |
| Number | `32-48px` | 700 | KPI数値・粗利数字の強調 |

### 3.3 強調ルール

- **太字 + Accent色** で重要キーワードをハイライト（例：`年1,600万円の粗利`）
- 黄色マーカー（`background: #FFF3A0`）は手書きメモ感を出すときに限定使用
- 引用は `❝` 記号 + Accent色のボーダー左 + Accent薄背景

## 5. コンポーネント

### 5.1 標準カード（Sub背景）

```css
background: #F2F2F2;
border-radius: 4px;
padding: 24px;
```

ヘッダー部に黒帯（`#1A1A1A`）+ 白文字でカテゴリ名を入れる。

### 5.2 警告カード（Warning）

```css
background: #FCE8E7;       /* Warning薄 */
border-top: 4px solid #E04E49;
border-radius: 4px;
```

「課題③」「機会損失」など赤ヘッダー帯で警告を明示。

### 5.3 引用ボックス（Accent）

```css
background: #E0F7F6;       /* Accent薄 */
border-left: 4px solid #0EBAB5;
padding: 16px 24px;
```

`❝` 記号 + 太字キーワード + 問いかけ文。

### 5.4 ステップアイコン（真円バッジ）

```css
width: 64px;
height: 64px;
border-radius: 50%;
background: #1A1A1A;       /* または Accent */
color: #FFFFFF;
font-size: 14px;
font-weight: 700;
```

`01 / 02 / 03` 形式の番号ラベルを真円で表現。

### 5.5 KPI数値ブロック

```css
font-size: 48px;
font-weight: 700;
color: #0EBAB5;            /* または #1A1A1A */
```

「1,200+」「4.7」「30+」のような実績数字に使用。

### 5.6 フッター帯（共通）

```css
background: #0EBAB5;
height: 8px;
width: 100%;
```

全ページ共通。これが入っているだけで一気に「コンサル資料感」が出る。

## 7. やってOK / NGリスト

### ✅ DO

- Accent色を全ページに必ず1箇所以上入れる（フッター帯で担保）
- 数字は大きく・太く・Accent色で
- 引用・問いかけは `❝` で柔らかく
- 余白を恐れずに使う
- タイトルスライドは黒背景で締める

### ❌ DON'T

- 紫グラデーションは絶対に使わない
- Accent色を本文中に多用しない（10%ルール）
- 明朝体・装飾系フォントは使わない
- 1スライドに3色以上のアクセントを入れない
- カード背景にAccent色を使わない（薄ティールならOK）
