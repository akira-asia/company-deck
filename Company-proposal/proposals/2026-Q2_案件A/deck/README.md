# deck/

最終提案書を格納するフォルダ。

## 推奨ファイル

- `final.pptx` … 最終提案書（クライアント提出版）
- `final.pdf` … PDF版（参考用）
- `outline.md` … 提案書の骨子・構成メモ（任意）

## 命名規則

- バージョン管理が必要な場合：`final_v1.pptx`, `final_v2.pptx`
- クライアント提出版は `final.pptx` に統一推奨

## 注意

- 機密情報を含む場合は、リポジトリのpublic/private設定に注意
- 大容量ファイル（>50MB）は Git LFS の利用を検討
