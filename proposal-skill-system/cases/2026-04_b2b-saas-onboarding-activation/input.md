# B2B SaaS オンボーディング改善提案 - Input

## 案件概要

- 業界: B2B SaaS
- BM型: SaaS サブスクリプション型
- 目的: 無料トライアルから有料化までの転換率を改善する

## 入力された分析メモ

- トライアル登録は安定しているが初回設定完了率が低い
- 営業、CS、プロダクトの責任範囲が分かれ、失注理由の粒度が揃っていない
- 有料化判断に必要な価値実感までの導線が長い

## Plan

- 既存の19枚構成をベースに、案件固有の転換ボトルネックへ置き換える
- KPIツリー、CS全景、施策詳細、ROIの整合性を最優先で確認する
- 骨子、HTMLDeck、PPTXの順でレビューを挟む想定にする
