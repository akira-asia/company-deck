# 地方ホテル 直販予約比率改善提案 - Input

## 案件概要

- 業界: 宿泊・観光
- BM型: 予約仲介依存型
- 目的: OTA依存を下げ、公式サイト経由の予約比率を改善する

## 入力された分析メモ

- OTA経由の予約比率が高く、手数料負担が利益を圧迫
- 公式サイトは情報量が多い一方で予約導線が弱い
- 季節需要と顧客セグメント別の訴求が整理されていない

## Plan
