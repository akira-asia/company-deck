# コスメD2C リピート購入率改善提案 - Input

## 案件概要

- 業界: コスメD2C
- BM型: F-1型 EC/D2C
- 目的: 初回購入後90日以内の2回目購入率を改善し、LTVを底上げする

- CRM施策はあるが顧客セグメント別に出し分けできていない
